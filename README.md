# CalendarApp

## Autor
Costea Andrei Ioan

## Description
This is a scheduling app made for a highschool project

## Functionalities
This app have a login system that verifies if the user (student) is logged in the database. 
The accounts are created by the admin directly in the db.

Students can organize their schedule in the calendar page by creating their own events or dragging them
on the calendar.

On the dashboard page every student can see the advices that other students posted. Also every student 
can post an advice.

## Technologies used
Python, Flask, mysql, sqlalchemy

## Mentions
The design is made with a prebuild template

## How to run
Run db.sql in your SQL editor.

Make sure to configure the database credentials and name in the main.py file. 

python ./main.py
